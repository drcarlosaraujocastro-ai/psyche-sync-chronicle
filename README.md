# PsicoNorte Clinical Sync

Aplicativo clínico-farmacológico determinístico: pacientes, sessões, doses, curvas PK/PD relativas, fenomenologia, interações, estoque, auditoria, conhecimento farmacológico e relatórios.

Este pacote remove IA externa/IA externa para reduzir custo, build quebrado e dependência de backend. A inteligência é local/determinística e usa os dados estruturados do Supabase.
